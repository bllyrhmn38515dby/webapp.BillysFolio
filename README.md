## 🚀 Deployment

Website ini di-deploy menggunakan Firebase Hosting. Untuk deploy versi Anda sendiri:

### Persyaratan
- Node.js dan npm terinstall
- Akun Firebase
- Firebase CLI terinstall

### Langkah Deployment

1. **Clone repository ini**
   ```bash
   git clone https://github.com/username/billys-folio.git
   cd billys-folio/static-site
   ```

2. **Install Firebase CLI**
   ```bash
   npm install -g firebase-tools
   ```

3. **Login ke Firebase**
   ```bash
   firebase login
   ```

4. **Inisialisasi project Firebase**
   ```bash
   firebase init hosting
   ```
   - Pilih project yang sudah ada atau buat yang baru
   - Set public directory ke `.`
   - Konfigurasi sebagai SPA: Yes
   - Tidak perlu overwrite file yang ada

5. **Deploy ke Firebase**
   ```bash
   firebase deploy
   ```

## ✨ Overview

**Billiesfolio** is a high-end, premium professional portfolio built with Laravel and styled with a cutting-edge **Antigravity-inspired Liquid Mesh Design**. It is designed to showcase technical expertise, project management capabilities, and system analyst skills with a focus on immersive user experience and clean, interactive UI.

## 🚀 Key Features

- **🌊 Liquid Mesh Background**: An interactive, fluid gradient system inspired by `antigravity.google` that reacts dynamically to mouse movement.
- **💎 Glassmorphism UI**: A consistent, modern design system featuring frosted glass effects, subtle borders, and deep shadows.
- **🎨 Custom Project Mockups**: High-fidelity product showcases for key projects like **eSIR 2.1**, **SIMAPAN**, and **SIMPU**.
- **📊 Interactive Analytics**: Skills represented through animated circular progress charts and data-driven snapshots.
- **📧 Seamless Contact Integration**: Fully functional contact form integrated with **Formspree** for AJAX-powered real-time messaging.
- **📱 Responsive Excellence**: Perfectly optimized for Desktop, Tablet, and Mobile devices.

## 🛠️ Tech Stack

- **Backend**: [Laravel](https://laravel.com/) (PHP)
- **Frontend**: Bootstrap 5, Vanilla JavaScript (ES6+), CSS3 (Flexbox/Grid)
- **Design**: CSS Liquid Mesh, Glassmorphism, Intersection Observer Animations
- **APIs**: [Formspree](https://formspree.io/) (Contact Integration)
- **Fonts**: [Inter](https://fonts.google.com/specimen/Inter)

## ⚙️ Development Setup

   ```bash

   ```

2. **Install dependencies**:
   ```bash

   npm install
   ```

3. **Configure Environment**:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Run Local Server**:
